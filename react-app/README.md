# Iluminação & Acabamentos

Projeto React/Vite baseado no visual do `Landing.html`, mantendo a aparência do protótipo e concentrando os ajustes em responsividade.

## Como rodar

```bash
npm install
npm run dev
```

Build de produção:

```bash
npm run build
```

## Onde editar

- Conteúdo, links, mensagens e fotos: `src/data/site.js`
- Estrutura das seções: `src/App.jsx`
- Visual e responsivo: `src/index.css`
- Fotos públicas: `public/images/`

## Deploy

Na Vercel, use:

- Build command: `npm run build`
- Output directory: `dist`
